# KNK Labs

Official web experience for KNK Labs — a futuristic product and systems showcase featuring the P.O.P Glove collection and other experimental wearable technologies.

Built with Next.js, Tailwind CSS v4, Motion, Supabase, and Resend.

---

## Overview

KNK Labs is a concept-driven digital experience focused on:

- Product storytelling
- Interactive configurators
- Futuristic wearable systems
- Premium visual presentation
- Waitlist and preorder infrastructure

The current flagship product is the **P.O.P Glove** — a driver-focused glove system integrating the AP x Swatch Royal Pop directly into the driving experience.
